1. LXDE(Lightweight X11 Desktop Environment) 羽量級桌面環境
    Fred 為 LXDE 的主要開發者之一，該計畫目標為創造一個輕量化之桌面環境，甚至可以適用在資源受限制的裝置，如：易PC(eeepc)、手機。
    * 專案網站：http://www.lxde.org/
  2. LXTerminal 輕量級的終端機模擬器
    Fred 是此專案的作者，此專案是一個 VTE-based 的終端機模擬器，其盡可能減少不必要的套件依賴，且不相依特定桌面環境。
  3. LXNM/Netstat Plugin for LXPanel 網路連線管理器
    Fred 是此專案的作者，這是一個快速、輕量且可獨立運作的網路連線管理器。
  4. OXIM 中文輸入法
    一個由最早期中文輸入方案，演變而來的中文輸入法架構，Fred 曾提交 Patch 給上游開發者。
  5. Frexhwd 硬體自動偵測程式
    這是 Fred 早期的一個專案，用來自動偵測硬體，並視情況自動掛載驅動程式和相關系統設定的工具。使用 Unix Shell(Bash) 和 C 語言所撰寫，易於使用在小型，嵌入式或簡易的系統之上。
  6. Bootchart-lite
    針對 Bootchart 的問題，為嵌入式系統所重新開發的版本。用來收集所有開機資訊以產生圖表，分析開機情形和速度。 * 專案網站：http://code.google.com/p/bootchart-lite/
  7. Openmoko - Open Source Mobile Phone 開放手機
    Fred 曾經協助減少 Neo FreeTunner 的開機時間。
  8. PostgreSQL
    Fred 為 PostgreSQL 中文社群作者之一。
    * 專案網站：http://postgresql-chinese.blogspot.com/
  9. PenMount 6000 Touchscreen Driver for kdrive/tslib:
    為 kdrive(TinyX on Embedded System) 所開發的 PenMount 6000 觸控螢幕的驅動程式，它是一個 tslib 的 Plugin。
    * 專案網站：http://people.linux.org.tw/~fred/drivers/pm6000_tslib.tgz
